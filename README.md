# Tweeebs

Embryo stage of a clone of the notorious social media "X", previously "Twitter". The idea is to make it tailored for anime/manga lovers. 

## Getting Started

These instructions will help you set up and run the project on your local machine.

### Prerequisites

Before you begin, make sure you have the following installed:

- [Node.js](https://nodejs.org/) - JavaScript runtime
- [npm](https://www.npmjs.com/) or [Yarn](https://yarnpkg.com/) - Package managers
- [Expo CLI](https://docs.expo.dev/get-started/installation/) - Expo Command Line Interface

### Installation

1. Clone the repository:

git clone https://github.com/your-username/your-react-native-app.git

2. Navigate to the project folder:

cd your-react-native-app

3. Install dependencies:

npm install
 or
yarn install

### Running the app
Start the Expo development server:

npx expo start

Running on an Emulator or Physical Device
Scan the QR code from the Expo DevTools using the Expo Go app on your mobile device.
or
Press i to open the app in the iOS simulator (macOS only) or a to open the app in the Android emulator.
